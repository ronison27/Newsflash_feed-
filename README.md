# ⚡ FlashFeed — Real-Time News App

A fast, mobile-friendly news app that delivers real-time headlines from around the world.  
Built with HTML, CSS, and Vanilla JavaScript — no frameworks needed.

## ✨ Features
- 🔥 Live news powered by GNews API
- 📂 Category filters — Top, Business, Tech, Sports, Health, Science, Entertainment
- 🔍 Search news by keyword
- 🔖 Bookmark articles to read later
- 🌙 Dark mode with saved preference
- 📺 Breaking news ticker with live headlines
- 🌍 Country selector — India, US, UK, Australia & more
- 📤 Share articles natively or via clipboard
- 📱 Fully responsive — works on mobile and desktop
